# Dino-runner-game
# 🦖 Dinosaur Game in C

A simple console-based Dinosaur Game developed in **C** using the Windows Console API. Control the dinosaur, jump over obstacles, and try to achieve the highest score.

## 🎮 Features

- Console-based gameplay
- Simple jump mechanics
- Random obstacle generation
- Real-time score counter
- Game Over detection
- Lightweight and beginner-friendly C project

## 🛠️ Technologies Used

- C Programming Language
- Windows Console API
- GCC / MinGW Compiler

## 📂 Project Structure

```
Dinosaur-Game/
│── main.c
│── README.md
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Dinosaur-Game.git
   ```

2. Navigate to the project folder:
   ```bash
   cd Dinosaur-Game
   ```

3. Compile the program:
   ```bash
   gcc main.c -o dinosaur
   ```

4. Run the executable:
   ```bash
   dinosaur.exe
   ```

## 🎮 Controls

| Key | Action |
|------|--------|
| Spacebar | Jump |
| ESC | Exit Game |
